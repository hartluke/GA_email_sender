General Accounting Email Sender

This application is an automated email sending tool. It will send emails based on the data extracted from the input spreadsheet.

This will output a logging file that shows operations performed by the script, along with timestamps and any errors that occur during runtime.

Please ensure that you have changed the default account in Outlook to the account you wish to send these emails from.

  • Input Report: The Excel file containing the data to be sent in the email.
  • Log File Output Directory: This is where the logging file will be output.

Once you've filled out all the necessary fields, click the 'Run' button to execute the script.

If you encounter any errors while running, please ensure that all required fields are filled out correctly. If the problem persists, email your log file to lucashart@mines.edu for further assistance.
